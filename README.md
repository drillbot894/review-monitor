# Review Monitor

Track review counts and ratings for your clients' GBP listings over time.

## Quick Start

```bash
# Open in browser
open ./review-monitor.html
```

## What It Does

- Track multiple business listings
- Monitor review counts over time
- Track rating changes
- Visualize review velocity trends
- Alert when new reviews come in

## Usage

1. Add client businesses to track
2. Set check frequency (manual for now)
3. View trends and velocity
4. Get alerts on new reviews

## Note

This version uses simulated data. To make it live:
- Use GBP API (limited access) 
- Or scrape with tools like gmbeverything
- Or use BrightLocal/Moz Local APIs
- Manual check option always works!
